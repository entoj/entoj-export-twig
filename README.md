# Entoj Twig Export

## Running tests

### Run all test specs at once

```
npm test
```

### Run all tests matching the given regex

```
npm test -- --grep model/
```

### Enable logging while running tests

```
npm test --vvvv
```

### Run all tests and shows test coverage

```
npm run coverage
```

### Lint all source files

```
npm run lint
```
